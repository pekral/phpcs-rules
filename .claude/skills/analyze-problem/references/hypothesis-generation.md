# Hypothesis Generation

## Goal
Generate multiple plausible explanations.

## Avoid
- First idea bias
- Confirmation bias

## Categories

### Code
- logic bug
- condition error

### Configuration
- env variables
- feature flags

### Infrastructure
- CI/CD
- network

### Data
- invalid input
- missing data

## Technique
- Always generate at least 2–3 hypotheses
- Prefer competing explanations
